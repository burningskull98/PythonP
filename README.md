Интернет-магазин игр для компьютера и PlayStation

Этот проект разработан с использованием Django и предоставляет пользователям возможность просматривать и покупать игры.

Реализовано:
Просмотр каталога игр
Добавление игр в корзину
Оформление заказов
Управление учетными записями пользователей
Административная панель для управления играми и заказами
Оплата товара

Установка
Клонируйте репозиторий; Создайте виртуальное окружение и активируйте его;
Установите зависимости; Запустите миграции и загрузите данные в БД; 
Создайте администратора магазина; Запустите сервер; 
Теперь вы можете открыть браузер и перейти по адресу http://127.0.0.1:8000/, чтобы увидеть интернет-магазин.

Использование Для просмотра игр перейдите на главную страницу. 
Чтобы добавить игру в корзину, нажмите кнопку "Добавить в корзину" на странице игры. Для оформления заказа перейдите в корзину и следуйте инструкциям.

Используемые технологии: Этот проект создан с использованием следующих технологий:
Python
